## 📝 开发规范

### 添加新功能时的多语言支持流程

1. **创建语言包文件**
   ```bash
   # 在对应的语言包目录创建新文件
   touch src/locales/zh-CN/your-module.ts
   touch src/locales/en-US/your-module.ts
   ```

2. **编写语言包内容**
   - 保持中英文语言包结构完全一致
   - 使用有意义的键名（camelCase）
   - 使用嵌套对象组织相关文本

3. **在语言包主入口中导入**
   ```typescript
   // src/locales/zh-CN/index.ts
   import yourModule from './your-module'
   export default {
     // ...
     yourModule
   }
   ```

4. **在组件中使用**
   ```vue
   <template>
     <div>{{ $t('yourModule.title') }}</div>
   </template>
   
   <script setup lang="ts">
   import { useI18n } from 'vue-i18n'
   const { t } = useI18n()
   </script>
   ```

5. **测试多语言**
   - 切换语言测试功能
   - 确保英文翻译准确、自然
